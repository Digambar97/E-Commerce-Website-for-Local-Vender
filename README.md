# E-Commerce-Website-for-Local-Vender

# 🛒 E-Commerce Multi-Vendor Platform

A scalable **multi-vendor e-commerce platform** built with **Java, Spring Boot, JSP/Servlets, and MySQL**.  
This project digitizes local storefronts, enabling vendors to list products, manage orders, and customers to shop seamlessly.

---

## 🚀 Features
- Multi-vendor support (local shops can register and sell online).
- Secure shopping cart and checkout system.
- Automated order management (confirmation, tracking).
- Optimized database queries for fast product catalog retrieval.
- Role-based access (Admin, Vendor, Customer).
- Responsive UI with Bootstrap for smooth navigation.

---

## 🛠️ Tech Stack
- **Backend:** Java 8, Spring Boot, Hibernate, JDBC, Servlet, JSP
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Database:** MySQL (RDBMS)
- **Tools:** Eclipse IDE, Git/GitHub, STS-4
